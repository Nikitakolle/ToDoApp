# ToDoApp

## Overview  
ToDoApp is a secure web application featuring user authentication and authorization built with ASP.NET MVC and Entity Framework. It allows users to register, log in, and manage their personal to-do tasks efficiently.

## Features  
- 🔐 User registration and login with authentication and authorization  
- ✅ Task management with ability to add, update, and delete tasks  
- 🗂️ Organized task tracking with intuitive UI  
- 🎨 Responsive design using Bootstrap for seamless experience on all devices

## Technologies Used  
- C#  
- ASP.NET MVC  
- Entity Framework  
- HTML, CSS, JavaScript  
- Bootstrap

## Getting Started  

### Prerequisites  
- .NET SDK (compatible version)  
- SQL Server or compatible database  

### Installation  
1. Clone the repository
git clone https://github.com/Nikitakolle/ToDoApp.git

2. Navigate to the project folder
cd ToDoApp

3. Configure your database connection string in appsettings.json

4. Run the application
dotnet run
